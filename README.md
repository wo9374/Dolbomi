# Dolbomi

돌보미는 유치원과 학부모 간 커뮤니케이션 앱 입니다.

교사와 학부모 간 채팅, 게시판을 통한 정보 공유, 출석부 등하원 푸시알림, 학원차량의 위치 등을 실시간으로 제공하는 앱 입니다.
